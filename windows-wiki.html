Installing Windows


Preparing a bootable media



Before we start the installation, we need to prepare a bootable media first. You need:

A USB drive, at least 8GB in size

A 2nd computer that runs Windows

The target computer (the computer in which the new windows will be installed.)

Now that you've prepared the following three things above, lets get started.
First of all, we need to download media creation tool, a software made by microsoft to upgrade your PC, and create windows bootable media. You can get this piece of software from microsoft's official website. You can get them here.

Once you've landed on the website, you choose wether you want to install Windows 10, or Windows 11 (note that windows 7, 8, and 10 has reached its end of life/EoL.) Download the Media Creation Tool software from the "Create a bootable media" section. It is referred by the image below.

Once you've downloaded the media creation tool software, double-click it on file explorer/click it once if opening from the browser download section, accept the agreement, and then choose your desired language and edition. After clicking next, make sure that you choose USB Flash drive from the selection, and make sure that your USB drive is plugged in. Select your USB drive from the list of the USB drive and click next. Now, wait for a few minutes for the software to download Windows 11 ISO, and flash it to your USB drive.

After your USB flash drive is ready, close the software, and safely eject your USB drive from your computer.




Installing Windows to a Computer/Disk



Now that you've prepared a bootable media for your new Windows PC, plug it in to your new target PC. When you turn it on, the boot order should automatically configure to boot into the installation media. If not, then refer to the steps below on how to configure boot order.

1. Turn on your computer, and repeatedly press F2, or ESC (depending on your motherboard's manufacturer)

2. Go to boot section of your BIOS, and change the boot order. The label "UEFI: USB" or similar should be on the top of the boot order. Example is with the image below.


3. After you configured the boot order, you can restart to apply the changes. And once you restarted, the computer will automatically boot to windows installation.



Starting installation to disk



Once you are in the installation media, you can now continue on the installation. Just click the install thingy and choose the default configuration for everything, unless you want a different language windows will show. After you've landed in the disk partitioning part, follow the steps below.


Delete every disk partition except your USB device's partition., and then select the unallocated space, then click next. Windows will now be on one partition.

After you've done all the steps above, now patiently wait for windows to be installed in your disk.
